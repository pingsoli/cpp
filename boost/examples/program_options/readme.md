# Boost program_options

+---------------------------+-------------------------------+
| symbol                    | description                   |

| options_description       | describes a number of options |
| value                     | define the option's value     |
+---------------------------+-------------------------------+
| Parser component                                          |
+---------------------------+-------------------------------+
| parse_command_line        | simpified interface           |

| parse_config_file         | parse config file             |
| parse_environment         | parse environment             |

| Storage component                                         |
+---------------------------+-------------------------------+
| variables_map             | storage for option values     |
